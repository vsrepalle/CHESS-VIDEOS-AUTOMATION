# ♟️ Chess Pro Automation Guide

## 🚀 How to Run
1. Edit `fen_short_puzzle/my_puzzle.json` for manual puzzles.
2. Run `run_chess_pro.bat` and select Option 2.

## 🧹 Built-in Force Cleaner
The system now automatically wipes `rendered_boards` before every run to prevent "ghost" images from old puzzles.

## 📂 Output
Videos are saved to: `C:\VISWA\CHESS_PRO_AUTOMATION\output\videos\`
